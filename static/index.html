Addon de Stremio

Este servidor expone un addon compatible con Stremio. Puedes instalarlo en Stremio usando la URL del manifest.

Ver manifest.json Abrir en Stremio

Para instalarlo en Stremio:

  1. Abre Stremio y ve a Addons.
  2. Elige Install via URL.
  3. Pega la URL de este manifest: http://localhost:7000/manifest.json (o usa el botón "Abrir en Stremio" para abrir la app directamente).

Endpoints disponibles: /manifest.json, /catalog, /meta, /stream. Este landing es estático por seguridad y simplicidad.
